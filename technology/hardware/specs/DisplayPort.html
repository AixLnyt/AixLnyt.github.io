DisplayPort 傳輸介面

2025-07-17
未分類
3k 字

DisplayPort（簡稱 DP）是由 VESA（Video Electronics Standards Association） 所制定的數位視訊傳輸標準，自 2006 年推出以來廣泛應用於電腦顯示器、專業繪圖卡、筆記型電腦、工作站、USB-C 介面顯示輸出等領域。
相較於 HDMI，DisplayPort 具備更高的頻寬、更彈性的多螢幕擴充能力（MST）、以及更早期導入高更新率與高解析度的支援，特別受到電競、創作者與工程專業族群青睞。

DisplayPort 各版本比較表

版本 發布年 最大解析度與更新率 傳輸頻寬（單向） 技術特性與備註
1.1 2007 4K@30Hz 8.64 Gbps 初代版本，適用於基礎外接顯示器
1.2 2010 4K@60Hz 17.28 Gbps 加入 MST（多串流傳輸），可一條線串接多顯示器
1.3 2014 5K@60Hz / 4K@120Hz 32.4 Gbps 引入 HBR3，為高更新率顯示鋪路
1.4 2016 8K@60Hz（使用 DSC） 32.4 Gbps 支援 HDR10、DSC（視訊壓縮），主流規格之一
2.0 2019 10K@60Hz / 16K@60Hz（使用 DSC） 80 Gbps 全面升級至 UHBR20，適用超高解析度顯示與多螢幕架構
2.1 2022 同上 80 Gbps 加強實體層穩定性，全面支援 USB4 架構整合
2.1a 2023 同上 80 Gbps 新增 Panel Replay，降低靜態畫面功耗，強化筆電應用



HBR（High Bit Rate）系列

名稱 每通道速率 總頻寬（4 通道） 對應版本 最大解析度 備註
RBR 1.62 Gbps 6.48 Gbps DP 1.0 1080p @30Hz 長度是2米，已經廢棄
HBR 2.7 Gbps 10.8 Gbps DP 1.0 / 1.1 DP 1.0 2K @60Hz / DP 1.1 4K @30Hz 最早支援 1080p @60Hz，目前極少使用
HBR2 5.4 Gbps 21.6 Gbps DP 1.2 4K @60Hz 可支援 4K @60Hz，多螢幕
HBR3 8.1 Gbps 32.4 Gbps DP 1.3 / 1.4 4K @120Hz / 8K @30Hz DP 1.4 + DSC 可到144Hz / 有 DSC 可到60Hz

補充說明：

  • HBR3 是目前 DP 1.4 主流所使用的傳輸模式。

UHBR（Ultra-High Bit Rate）系列【DP 2.0/2.1專屬】

名稱 每通道速率 總頻寬（4 通道） 最大解析度 備註
UHBR10 10.0 Gbps 40 Gbps 4K @ 144 Hz；1440p @ 354 Hz；1080p @ 555 Hz 實際就像 DX12、遊戲主流 4K@144Hz 畫質的安全選擇
UHBR13.5 13.5 Gbps 54 Gbps 4K @ 81 Hz；1440p @ 452 Hz；1080p @ 688 Hz 應用於高刷新 OLED 顯示器（如 8K@240Hz）
UHBR20 20.0 Gbps 80 Gbps 4K @ 323 Hz；8K @ 63 Hz；1440p @ 609 Hz；1080p @ 884 Hz 技術極限，多數環境仍依賴 DSC；支持超高解析度格式

補充說明：

  • 表中數值來自標準 CVT‑RB v2 timing 計算，不包含 DSC（Display Stream Compression）。
  • 實際使用中，螢幕與 GPU 端常有刷新率或 DSC 限制；例如 UHBR10 主流線材更常支援 4K@144Hz 而非極限計算值。
  • UHBR 是 DP 2.0 / 2.1 的關鍵升級點，支援極高解析度（如 16K@60Hz）、多路輸出或極高更新率（如 4K@240Hz）。



DisplayPort 長度建議與選線說明

標準版本 最大傳輸速率 建議／認證長度 備註
DP 1.4 32.4 Gbps ≤ 3 公尺（VESA 認證） 支援 4K@120Hz / 8K@60Hz，仍是目前主流
DP 2.0 / 2.1 80 Gbps（UHBR20） ≤ 1.5 公尺（被動線） 頻寬極高，建議使用光纖 DP（AOC）或主動式線材延伸距離

補充說明：

  • UHBR（Ultra-High Bit Rate） 為 DP 2.x 採用的新數據通道，提供 10～20 Gbps 每通道（總四通道為 80 Gbps）
  • MST（Multi-Stream Transport）：可透過一個 DP 埠輸出多螢幕畫面，常用於串接雙螢幕或多螢幕 dock。
  • DSC（Display Stream Compression）：無損壓縮技術，用於支援高解析度（如 8K、10K）顯示輸出時減少頻寬壓力。



HDMI vs DisplayPort vs DVI 對比分析

項目 HDMI DisplayPort (DP) DVI
發表時間 2002 2006 1999
音訊整合 ✅ 一條線整合視訊與音訊 ✅ 可整合音訊（部分版本或透過 USB-C） ❌ 僅支援視訊
頻寬上限 48 Gbps（HDMI 2.1） 80 Gbps（DP 2.0 / 2.1） 3.96 Gbps（DVI-D Dual Link）
更新率支援 最高 4K@120Hz / 8K@60Hz 最高可達 16K@60Hz（DSC） / 4K@240Hz 通常限制於 1080p@60Hz
多螢幕輸出 ❌ 不支援 MST ✅ 原生支援 MST ❌
傳輸接頭類型 A / C / Micro（非鎖定式） 標準 DP / Mini DP / USB-C Alt Mode DVI-D / DVI-I（已過時）
主流應用 電視、音響、遊戲機 電腦、工作站、顯示器、USB-C 擴充底座等 老舊投影機、工控螢幕
可攜性與整合性 中（線長與方向受限） 高（支援轉 USB-C、多設備整合） 低

省流：

  • DVI：過時
  • HDMI：消費導向、影音設備、注重相容性
  • DP：專業導向、擴充性、效能更好



補充技術名詞 MST / DSC / Panel Replay

名稱 全名 說明
MST Multi-Stream Transport 一埠多螢幕輸出（如 Daisy Chain 串接）
DSC Display Stream Compression 類視覺無損視訊壓縮，降低頻寬佔用，支援高解析度
Panel Replay Panel Replay 類似增量畫面刷新，減少靜態畫面功耗，提升筆電效能與續航



DisplayPort（VESA 標準）

標準版本 最大傳輸速率 建議／認證長度 備註
DP 1.4 32.4 Gbps ≤ 3 公尺（VESA 認證） 支援 4K@120Hz、8K@60Hz
DP 2.0 / 2.1 80 Gbps（UHBR20） ≤ 1.5 公尺（被動線） 高頻率傳輸須控制線長；長距離建議採光纖線



DisplayPort 與 USB-C 整合趨勢

現代筆電與行動工作站多採用 USB-C（支援 DisplayPort Alt Mode） 作為顯示輸出通道，
搭配 Thunderbolt 4 或 USB4 控制器時，USB-C 可提供：

  • 最高 40 Gbps 傳輸速率

  • 4K@60Hz 或更高解析度影像輸出

  • PD 快充 + 視訊 + 資料三合一輸出

    建議使用 VESA 認證 的 USB-C to DP 或 USB-C to HDMI 線材



DisplayPort 選購建議

  • 專業圖形 / 電競：DP 1.4 以上，短線或光纖線材，支援 4K@120Hz、8K@60Hz（DSC）
  • 筆電外接 / 多螢幕：DP with MST 或 USB-C Alt Mode
  • 長距穩定傳輸：主動式 DP 線 或 光纖 DP（AOC）
