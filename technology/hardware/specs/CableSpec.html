DisplayPort / HDMI / USB 傳輸線材建議長度對照表

2025-07-15
未分類
888 字

本表彙整常見高頻寬數位線材（DisplayPort、HDMI、USB/PD）之標準版本、對應傳輸速率與建議長度，供選購及佈線配置時參考。


DisplayPort（VESA 標準）

標準版本 最大傳輸速率 建議／認證長度 備註
DP 1.4 32.4 Gbps ≤ 3 公尺（VESA 認證） 支援 4K@120Hz、8K@60Hz
DP 2.0 / 2.1 80 Gbps（UHBR20） ≤ 1.5 公尺（被動線） 高頻率傳輸須控制線長；長距離建議採光纖線



HDMI（HDMI Forum / HDMI LA）

標準版本 最大傳輸速率 建議／認證長度 備註
HDMI 1.4 10.2 Gbps 約 5～10 公尺 支援 1080p、4K@30Hz
HDMI 2.0 18 Gbps 約 3～5 公尺 建議使用 Premium HDMI 線
HDMI 2.1 48 Gbps ≤ 3 公尺（認證建議） 建議使用 Ultra High Speed HDMI 線材



USB / PD 快充與傳輸線

標準版本 最大傳輸速率 建議／認證長度 備註
USB 2.0 480 Mbps 約 3～4 公尺 適用低速設備與基本充電
USB 3.2 Gen 1 / Gen 2 5 / 10 Gbps 約 1～2 公尺 傳輸速率越高，建議越短越穩定
USB4 / Thunderbolt 4 40 Gbps ≤ 0.8 公尺（被動）
≤ 2 公尺（主動）		 被動線長度有限，超出須使用主動線避免降速
USB PD 100W（20V/5A） – ≤ 1～2 公尺（需帶 E-Marker） 超過 2 公尺通常會自動降至 60W 以下
USB PD 140W（28V/5A） – ≤ 1 公尺（EPR 等級線） 需使用支援 EPR 的 Type-C to C 線材（帶 e-Marker）
USB PD 240W（48V/5A） – ≤ 1 公尺（EPR 等級線） 線材與裝置均需支援 EPR 規範，現市售仍不普及

補充說明

  • EPR（Extended Power Range） 是 USB PD 3.1 新增的高功率輸出範圍，支援高於 100W 的功率（最多 240W）。
  • E-Marker 晶片 為線材中內建的晶片，用以識別線材是否支援高電流或高速傳輸，140W / 240W 線需強制具備。
