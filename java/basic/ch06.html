常見的內建類別一覽

類別名稱 所屬套件 用途說明
String java.lang 字串處理
Math java.lang 數學計算
Scanner java.util 輸入處理（鍵盤輸入）
Random java.util 產生亂數
Date java.util 表示日期時間（較舊）
LocalDate java.time 處理日期（較新）
ArrayList java.util 動態陣列（集合類別）

String 類別

方法 說明
.length() 回傳字串長度
.charAt(i) 取得第 i 個字元
.substring() 截取子字串
.equals() 比較內容是否相等
.equalsIgnoreCase() 忽略大小寫比較
  • 注意：== 是比較記憶體位置，不可用來比較字串內容！

Math 類別

方法 說明
Math.abs() 絕對值
Math.pow(x,y) 次方運算
Math.sqrt() 開根號
Math.max() 取最大值
Math.min() 取最小值
Math.round() 四捨五入

Scanner 類別

方法 資料型態
nextLine() 讀取一整行字串
nextInt() 整數輸入
nextDouble() 小數輸入
next() 讀一個單字

Random 類別

方法 說明
nextInt(n) 0 ~ n-1 的亂數整數
nextDouble() 0.0 ~ 1.0 亂數小數
nextBoolean() true 或 false 隨機

LocalDate 類別（新日期類別）

方法 說明
LocalDate.now() 取得今天日期
plusDays(n) 加上 n 天
minusMonths(n) 減去 n 個月

記憶體觀念（stack vs heap）

  • 內建類別（如 String, Scanner）大多是物件型態（參考資料形態），儲存在 heap（堆積區）。

  • 宣告的參考變數（如 String s = "Hi";）則存於 stack（堆疊區）。

常見錯誤與迷思

錯誤類型 錯誤範例 原因說明
== 比較字串 "abc" == "abc" 檢查的是位置，不是內容
忘記 import Scanner 無法使用 忘了引入 java.util.Scanner
錯用 next() next() 只能輸入一個字 輸入多字需用 nextLine()
