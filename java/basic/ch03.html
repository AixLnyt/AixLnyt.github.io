基本輸出方法

Java 使用 System.out 物件來負責標準輸出（通常是畫面/終端機）。

print vs println

方法名 說明
System.out.print() 輸出內容，不換行
System.out.println() 輸出內容並換行

基本輸入方法（使用 Scanner 類別）

Java 沒有像 Python 那樣內建的簡單輸入函數，需要先引用 java.util.Scanner 類別。

使用步驟

  1. import java.util.Scanner;

  2. 建立 Scanner 物件 → Scanner sc = new Scanner(System.in);

  3. 呼叫對應方法讀取輸入內容（例如：nextInt()、nextLine()）

常見輸入方法與說明

方法 資料型態 範例輸入
next() 字串（不含空白） abc
nextLine() 字串（含空白） abc def
nextInt() 整數 123
nextDouble() 浮點數 12.34
nextBoolean() 布林 true 或 false

注意事項

  • nextLine() 和 next() 的差異：nextLine() 會讀取整行文字，包含空白；next() 只會讀到第一個空白前的字。

堆疊記憶體補充

雖然輸入/輸出主要是處理資料流，但使用 Scanner 宣告的物件屬於參考型態，儲存在 Heap 記憶體中，而基本型態的輸入值（如 int age）則存在 Stack 中。

常見錯誤

錯誤描述 原因說明與解法
InputMismatchException 發生 輸入的資料型態不符（如輸入文字給 nextInt()）
nextLine() 沒讀到資料 前面用了 nextInt()，未清除換行符號
輸入一直沒反應 沒有 System.out.print() 提示使用者輸入
