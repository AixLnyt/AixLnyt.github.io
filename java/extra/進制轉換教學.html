進制系統基礎概念

進制是數字表示法的基數系統，不同進制使用不同的基數來表示數值：

進制名稱 基數 使用數字 範例 程式表示
二進制 2 0, 1 1011 0b1011
八進制 8 0-7 377 0377
十進制 10 0-9 255 255
十六進制 16 0-9, A-F FF 0xFF

位值權重計算

每一位的權重 = 基數^位數（從右邊第0位開始）

錯誤處理與注意事項

常見例外處理

注意事項

  1. 進制範圍：radix 必須在 2-36 之間
  2. 字符限制：每個進制只能使用對應的有效字符
  3. 大小寫：十六進制的 A-F 不區分大小寫
  4. 溢位問題：超出 int 範圍會拋出例外
  5. 負數處理：支援負號前綴

位元運算與進制轉換



重點整理

  • 輸入轉換：使用 Integer.parseInt(string, radix) 將任意進制字串轉為十進制數值
  • 輸出轉換：使用 Integer.toString(int, radix) 系列方法將數值轉為指定進制字串
