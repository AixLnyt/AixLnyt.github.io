Java 教學目錄

2025-06-05
Java
658 字
  • 程式語言

Java 是一種跨平台的物件導向程式語言，常用於開發桌面應用、網站後端及 Android App。基本語法包括變數、條件判斷、迴圈、方法與類別等，進階可學繼承、多型、例外處理與資料結構。Java 擁有強大的標準庫，如 ArrayList、HashMap、Scanner 等，並支援檔案處理與多執行緒，是入門與進階都很實用的語言。
